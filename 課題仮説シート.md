# 課題仮説シート

## プロダクト名

メガファイル便

## 一言で言うと？

ファイル共有を指定した方のみに制限できるサービス

## 1. 誰の課題か？（ターゲットユーザー）

- **どんな人？**：今回のようなアイデアソンに参加される方々
- **人数の規模感**：アイデアソン参加人数
- **具体的に1人思い浮かべてください**：私

## 2. どんな課題か？

- **困っていること**：アイデアソンで提出するモックの管理が煩雑。
- **今どうやって対処しているか**：organizationにリポジトリを作って評価チームに権限を付与。
- **なぜそれでは不十分か**：GoogleDriveだと設定が面倒。GitHubだとリポジトリが煩雑になる。

## 3. なぜ今この課題を解くのか？

定常的に開催される可能性があるため。
今回実施にあたりファイル管理に関して実際に作れば良いという声が上がっていたため。

## 4. どう解決するか？

- **解決アプローチ（1-2文）**：Google連携を行ったアカウント管理とそれらに紐付くIDから閲覧権限を付与する。
- **モックで見せるコア機能**：権限付与者はフォルダにアクセスでき、されていない者はアクセスできない。
- **ユーザーがこのプロダクトを使ったら何が変わるか**：今後のアイデアソン実施にあたり提出方法がシンプルになる。
