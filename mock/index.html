GitHubでログイン
Personal Access Token を使ってGitHubアカウントに接続し、
実際のリポジトリのファイルを読み込みます。
1
GitHub にログインし Settings → Developer settings → Personal access tokens → Tokens (classic) を開く
2
「Generate new token (classic)」をクリック
3
スコープ repo にチェックを入れてトークンを生成
4
生成されたトークンをコピーして下に貼り付ける
