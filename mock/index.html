GitHubと連携
Personal Access Token (PAT) を入力してGitHubアカウントと連携します。
トークンはブラウザにのみ保存され、外部に送信されません。
1
GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic)
2
「Generate new token (classic)」をクリック
3
スコープ: repo・read:user にチェック
